**Goal:** Expand Agentic Architecture Atlas from five to fifteen complete case studies and make MDX front matter the single source of truth for a generated, filterable case catalog.

**Architecture:** Extract content discovery and front-matter parsing into a shared Node module, validate a strict catalog schema, and generate one deterministic JSON artifact consumed through a typed TypeScript adapter. Keep the existing five cases featured, add a compact 5/2/3 migration map to the homepage, and render all fifteen cases through a client-side catalog on the case-library page.

**Tech Stack:** Docusaurus 3.10.2, React 19, TypeScript 6, MDX, Mermaid, Node.js 24, Node test runner, CSS Modules, npm, GitHub Actions, GitHub Pages, GitHub CLI.
